# 我的技术博客 - GitHub Pages版

这是一个完全基于前端技术的静态博客系统，可以部署在GitHub Pages上，无需后端服务器。

## 功能特性

- 📝 **Markdown文章编辑** - 使用SimpleMDE编辑器，支持实时预览
- 🗂️ **文章分类管理** - 技术、教程、笔记、生活四大分类
- 🏷️ **标签系统** - 为文章添加多个标签
- 🔍 **文章搜索** - 支持按标题、内容、标签搜索
- 🌐 **网址导航** - 内置常用开发资源链接
- 💾 **本地存储** - 使用浏览器LocalStorage保存数据
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🚀 **一键部署** - 只需上传到GitHub即可使用
